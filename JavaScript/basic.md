# JavaScript

## 변수
  1. `let`    재할당 가능
  2. `const`  재할당 불가능
  3. `var`    과거에는 많이 사용했지만, 현재는 여러 단점들 때문에 사용 지양

## 데이터 타입
  1. 숫자
  ```console.log(10)
  const age = 20
  console.log(age)'''
  2. 문자열
  ```console.log('string')
  const string = 'poem'
  console.log(string)```
  3. Boolean
  ```console.log(true)
  const bool = false
  consol.log(bool)```
  4. null, undefined
  ```const name1 = null
  const name2
  console.log(name1)
  console.log(name2)```
