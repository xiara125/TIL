# JavaScript

## 변수
  1. `let`    재할당 가능
  2. `const`  재할당 불가능
  3. `var`    과거에는 많이 사용했지만, 현재는 여러 단점들 때문에 사용 지양

## 데이터 타입
  1. 숫자
  ```javascript
  console.log(10)       // 10
  const age = 20
  console.log(age)      // 20
  ```
  2. 문자열
  ```javascript
  console.log('string') // string
  console.log('30')     // 30 (문자열)
  const string = 'poem'
  console.log(string)   // poem
  ```
  3. Boolean
  ```javascript
  console.log(true)     // true
  const bool = false
  consol.log(bool)      // false
  ```
  4. null, undefined
  ```javascript
  const name1 = null
  const name2
  console.log(name1)    // null
  console.log(name2)    // undefined
  ```
